# Phoenix Nest — Python

Shared Python packages and utilities for the Phoenix Nest ecosystem.

Reserved for cross-module Python code (helpers, shared models, CLI tools) used by `flow/` and future modules.

## Setup

```bash
cd python
python -m venv .venv
.venv\Scripts\activate    # Windows
pip install -r requirements.txt
```
